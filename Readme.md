# 日常刷leetcode
算法确实是基础，锻炼思维能力，解决问题的能力

日常没事就可以做几道，等基础比较牢固以后，可以参加周赛，也是个不错的历练

# 为什么不在网页上直接写
1. 深度vim用户，网页上的操作还是有点麻烦，虽然也提供了vim的模式，但是只是简单支持了hjkl,很多命令都没有
2. 可以调试，可以记录自己的一些想法
3. 每个目录下可以放多个方法，每次都新写一个，看看是否有进步
4. 可以使用不同的语言实现

# Feature
* 集成了gtest，方便测试一些基础案例
* gen.sh可以生成对应题目的目录，以及部分代码，cmake的代码也生成了，可以更方便的专注于题目的书写
* common库提供了做题中需要用到的一些输入输出，转化，基础类型的定义，只要将题目中的Solution复制过来，就可以直接做题了。
